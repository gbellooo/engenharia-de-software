# UC003 – Atualizar Status de Incidente

**Aluno:** Gabriel Bello  
**RA:** 10416808 

## Ator Principal
Analista de Segurança

## Pré-Condições
- O analista deve estar autenticado no sistema.
- O incidente deve estar previamente registrado.

## Pós-Condições
- O status do incidente é atualizado corretamente.
- O histórico de alterações é registrado.

## Fluxo Principal
1. O analista acessa o sistema.
2. O analista seleciona um incidente existente.
3. O sistema exibe os detalhes do incidente.
4. O analista escolhe a opção de atualizar status.
5. O analista define o novo status (ex: "Em análise", "Resolvido").
6. O sistema valida a alteração.
7. O sistema atualiza o status do incidente.
8. O sistema registra a alteração no histórico.
9. O sistema exibe mensagem de sucesso.

```plantuml
